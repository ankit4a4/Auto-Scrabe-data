Enter the Category URL + Start/End Date (e.g. 1 July to 10 July). The system will crawl category pages and find every post between those two dates (based on each post's own publish date - if the category listing also shows dates, that speeds up pagination). If a post's date can't be determined, it is safely skipped rather than guessed. There's a safety limit on how many posts are processed in one run (for a very large range, only the first batch will be processed, and this will be noted in the log). Save condition: an entry is saved when EITHER Owner Name(s) OR Business Name is found (just one is enough - whichever field is missing stays blank). For partnerships with multiple owners, all names will be included. City, Phone, and Email are optional fields - they only appear if explicitly mentioned in the text. Non-business content (movies/entertainment/general news) is automatically filtered out.
